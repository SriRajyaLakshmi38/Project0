Technical Skills Required to be a Data Scientist:

Programming Languages Required

  1. Basic Libraries: NumPy, SciPy, Pandas, Ipython, matplotlib
  2. Libraries for Machine Learning: scikit-learn, Theano, TensorFlow

Statistics

  1. Descriptive Statistics
  2. Exploratory Data Analysis,Percentiles and Outliers
  3. Random Variables,Cumulative Distribution function (CDF),Skewness
  4. Probability Theory,Bayes Theorem

Data Bases

  1. SQL/ORACLE
  2. Column: Cassandra, HBase
  3. Document : MongoDB, CouchDB
  4. Key value: Redis, Dynamo

Soft Skills Required to be a Data Scientist:

  1. Data intuition
  2. Domain knowledge
  3. Research expertise
